## Overview

To use the Zedblock API, you need to generate an API key. This key is required for authenticating your requests and accessing various endpoints.

## Steps to Generate an API Key

1. Go to [app.zedblock.com](https://app.zedblock.com).
2. Log in with your credentials.
3. Navigate to **Settings** from the dashboard.
4. Under **API Keys**, you will find the option to generate a new API key.

Here's a screenshot of the dashboard showing the API key management:

![API Key Dashboard](https://ik.imagekit.io/1ze8oedlp/Business%20Profile%20-%20Google%20Chrome%2016-09-2024%2016_16_53.png)

Once you generate an API key, make sure to store it securely, as it will be needed to authenticate your API requests.

<Callout type="info">
  Store API key safely! Anyone who has your API key has full control over your
  account. For security reasons, you cannot view the API key again.
</Callout>

## Usage

You can include the API key in your request headers like so:

```bash
apikey: <your-api-key>
```
